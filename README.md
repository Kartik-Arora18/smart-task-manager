# ✅ Smart Task Manager

A Smart Task Manager web app built using AntiGravity multi-agent AI workflow with Claude. 4 agents were used: Planning, Frontend, Backend, and Integration Agent.

---

## 🤖 Agent Workflow (AntiGravity)

| Agent | Role |
|---|---|
| Planning Agent | Defined features, folder structure, and task data model |
| Frontend Agent | Designed HTML layout and CSS styling |
| Backend Agent | Wrote JavaScript logic and localStorage persistence |
| Integration Agent | Merged all code, tested, and fixed bugs |

---

## 🚀 Features

- Add tasks with title, priority and due date
- High / Medium / Low priority with colour-coded badges
- Mark tasks as complete (moves to Completed section)
- Undo completed tasks back to Pending
- Delete tasks
- Live search to filter tasks instantly
- Task counter — Total, Pending, Done
- Dark mode toggle (preference saved)
- localStorage — data persists on page refresh
- Mobile responsive design
- Overdue dates highlighted in red
